Sports Club Database System
Relational Database Design for Sports Club Management

Overview
This project simulates a real-world sports club management system using a relational database built with SQL. It is designed to handle all core operations of a sports club including employees, players, teams, facilities, subscriptions, payments, bookings, and cafeteria management.

Objectives

* Design a scalable relational database system
* Manage sports club operations efficiently
* Organize data for employees, players, teams, and members
* Implement relationships between multiple entities
* Ensure data integrity using keys and constraints
* Apply database normalization principles

Tools and Technologies

* SQL (MySQL)
* Relational Database Design
* ERD Concepts
* Primary and Foreign Keys
* Database Normalization

Key Features

* Employee management system (staff, coaches, departments)
* Player and team management
* Facility booking system
* Membership and subscription handling
* Cafeteria system with items and transactions
* Payment and pricing management
* Logging system for staff activities

Database Structure

* Employee and Job Management
* Player and Team Module
* Facility and Booking System
* Membership and Subscription System
* Cafeteria Management System
* Payment System
* System Logs
